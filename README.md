# JAKWO WARZONE - Boxing Ring + Dynamic Events Build

Run locally:

```bash
npm install
npm start
```

Open:

```text
http://localhost:3000
```

## Included

- Node.js + Express + Socket.IO multiplayer server
- 5000x5000 pixel open field
- 100 real player limit
- 50 troll NPC bots
- No account system: upload meme face + name + play
- Big meme head + tiny pixel body style
- Phone landscape controls: move joystick + manual aim joystick + attack/dash/chat buttons
- PC controls: WASD + mouse aim + click attack + space/dash button
- No auto aim
- Short range combat
- Boxing gloves
- Boxing rings: max 2 fighters inside, outside players cannot damage fighters inside
- 10 second spawn protection
- 1 starting life, loot extra lives up to max 3
- Loot, mystery boxes, weapons, flying board
- Loot despawns and respawns randomly
- Worm, bug, frog, snail traps
- World boss with 1000 HP and super power drop

## New Dynamic Events

- Meme Storm: temporary visibility chaos
- King of the Hill: temporary zone, players inside get small heal/speed boost
- Zombie Horde: temporary wave of zombies that chase and bite players
- Dance Fever Zone: temporary funny speed boost zone

The game is designed as a 24/7 persistent chaos battlefield. No rounds. No final winner. Players enter, fight, die, leave, and new players take their slots.
